# Pomodoro Timer Android App

A simple focus timer built with `Kotlin`, `Jetpack Compose`, and `MVVM architecture`.

The application helps users stay focused using timed work sessions and keeps a history of completed sessions.

## Overview

This app provides a clean timer experience inspired by the Pomodoro technique. Users can start a focus session, pause it, reset it, or finish it manually. Every completed session is saved locally and can be viewed later on the history screen.

## Features

1. Focus timer with start and pause controls
2. Reset or manually finish a session
3. Circular timer progress indicator
4. Local history of completed sessions
5. Settings and navigation support
6. Persistent storage using Room Database
7. Reactive UI powered by StateFlow

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3 UI components
- MVVM architecture
- Kotlin Coroutines and Flow
- Room Database

## Installation

1. Clone the repository:

```
git clone https://github.com/Viktoriia-code/pomodoro-timer-android.git
```
2. Open the project in Android Studio
3. Sync Gradle
4. Run the application on an emulator or physical device

## Requirements

1. Android Studio Hedgehog or newer
2. Android SDK 24 or higher
